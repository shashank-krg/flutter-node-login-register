```
git clone https://github.com/shashank-krg/flutter-node-login-register.git
```

`Change Directory`

```
cd flutter-node-mongodb-login-registration
```

`Change Directory`

```
cd backend
```

`Run Npm Command`

```
npm install
```

```
node index.js
```

`Change Directory`

```
cd frontend
```

`Run Command`

```
flutter run
```

`Or Open VSCode and`

```
Ctrl/Cmd + F5
```
